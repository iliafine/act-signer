# Сервис подписания актов

## Что это
Локальный сервис (FastAPI + браузерный UI) для автоматической вставки
сканов подписей в акты .docx. Работает офлайн, без реестра — просто
загружаете акт или zip с актами, сервис сам находит блок подписантов,
ищет нужные подписи в базе и вставляет их.

Отдельно — `registry_batch/batch_sign.py`: батч-режим по реестру (Excel),
который сам находит нужные папки актов по статусу «Принято» и подписывает
все сразу.

## Запуск сервиса

**Mac:** дважды кликнуть `start.command` (при первом запуске macOS может
попросить разрешить запуск в Системных настройках → Безопасность).
Откроется http://127.0.0.1:8743 в браузере.

**Windows:** дважды кликнуть `start.bat`. Требуется Python 3.10+
(python.org, при установке отметить «Add to PATH»).

**Остановка:** `stop.command` (Mac) / `stop.bat` (Windows).

## Структура

```
backend/            — FastAPI-приложение, логика подписания
  app/main.py        — веб-API
  app/docx_signer.py — поиск подписантов в акте + вставка подписи
  app/image_utils.py — очистка скана подписи (убирает фон/наложения)
  app/signature_db.py— база подписантов (signatures/db.json)
signatures/          — обработанные подписи (PNG) + db.json
signatures_raw/      — исходные сканы подписей (как пришли)
registry_batch/      — батч-режим по реестру (Excel)
test_data/           — тестовые файлы для проверки
```

## Текущий статус (MVP)
- Поддержка актов: **.docx** (PDF — в планах, пока не реализовано).
- Очистка сканов подписи: эвристика на OpenCV (адаптивная бинаризация +
  фильтр по компонентам). Работает на неоднородном фоне, но для сложных
  наложений (печать прямо по подписи) может требовать ручной donastройки
  через `crop_box` в `image_utils.extract_signature`.
- Поиск блока подписантов в акте: эвристика по таблицам/параграфам,
  проверена на синтетическом тестовом файле. **Нужно прогнать на реальных
  актах** и при необходимости донастроить `docx_signer.py`.
- Батч-режим по реестру: названия столбцов реестра и формат имени папки
  заданы по описанию, не по реальному файлу — нужна донастройка после
  получения реального zip (`registry_batch/batch_sign.py`, переменные
  `STATUS_COLUMN_CANDIDATES`, `FOLDER_COLUMN_CANDIDATES`).
